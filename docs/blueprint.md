# **App Name**: CampusLibro

## Core Features:

- Book Catalog: Browse and search for books by title, author, ISBN, or subject.
- User Authentication: Secure login and registration for students and librarians.
- Borrowing & Returns: Manage book loans, due dates, and returns.
- Search Assistant: An AI powered tool that helps you search for a book based on its topic by describing the book that you have in mind.
- Inventory Management: Track book availability and manage library inventory.
- Notifications: Send email reminders for overdue books.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and knowledge.
- Background color: Very light indigo (#E8EAF6), almost white, providing a clean and calm backdrop.
- Accent color: A muted blue-violet (#5C6BC0), for interactive elements and important information.
- Body and headline font: 'Inter' for a clean, modern, readable style suitable for long texts.
- Use simple, consistent icons for navigation and actions.
- Prioritize a clean and intuitive layout for easy navigation.
- Subtle animations to enhance user interaction (e.g., page transitions).
